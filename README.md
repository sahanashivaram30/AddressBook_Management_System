                                Address Book Management System.
A C-based Address Book application that allows users to create, list, search, edit, and delete contacts in an organized and interactive way. 
It ensures valid and unique contact details while providing a formatted display for easy access.

Features:
* Create Contact: Add new contacts with name, mobile number, and email.
* List Contacts: Display all contacts in alphabetical order in a formatted table.
* Search Contact: Search by name, mobile number, or email. Supports multiple matches and duplicates resolution.
* Edit Contact: Update name, number, email, or all details of an existing contact.
* Delete Contact: Permanently remove a contact from the address book.
* Load Contacts: Load saved contacts from contact.txt.
* Save Contacts: Save the current contacts to contact.txt.

Input Validation:
* Name: Only alphabetic characters and spaces are allowed.
* Mobile Number: Must be 10 digits, starting with 6, 7, 8, or 9.
* Email: Must be lowercase, follow proper format (example@domain.com), and unique.
